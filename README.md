# OpenCore Templates

Official repository for the [OpenCore Framework](https://github.com/newcore-network/opencore-cli) CLI templates.

This repository serves as a central hub for official templates. The OpenCore CLI automatically detects this repository to help you quickly download and integrate high-quality, pre-configured resources into your project.

## Quick Start

To clone a template into your project, use the following command:

```bash
opencore clone <template-name>
```

## Available Templates

### Resources
- **xChat**: A modern chat system with a classic SA:MP-inspired aesthetic. 
  - **Features**: Command history, inline color codes, auto-hide, and roleplay commands (`/me`, `/do`, `/ooc`, etc.).
